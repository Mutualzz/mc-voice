# Amecs compile stubs

These jars are the nested Amecs API modules extracted from
[Amecs 1.6.5 for MC 26.1](https://modrinth.com/mod/amecs) (`amecs-bundle`).
They are **compile-only** — not shipped in `mutualzz-voice`. Players still need
the full Amecs mod in `mods/` at runtime.

Refresh after bumping `amecs_version` in `gradle.properties`.
